Steganography

Steganography is the practice of concealing information within another message or physical object to avoid detection. Steganography can be used to hide virtually any type of digital content, including text, image, video, or audio content. That hidden data is then extracted at its destination.

Tools to be used: Stegosuite, dig TXT file , hex to Txt

  2. Download the image from the link given below.
  3. Use stegosuite to extract the hidden message.
  4. To extract password is required but jhonny is hiding password. Find it!
  5. Use hex to Txt to extract the hex code.
  6. Use dig TXT file to extract the hidden message.
